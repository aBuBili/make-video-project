## 在线预览
可以直接访问 http://pangtuo.vip 查看运行效果

## 下载项目
第一步  克隆项目代码到本地
```
git clone https://github.com/aBuBili/make-video-project.git
```

第二步 在项目根目录下 安装npm依赖
```
npm i
```
 
第三步 运行项目
```
npm run dev
```
或者
```
yarn dev
```

## 技术使用说明
- vite 2.0
- vue 3.0 (script setup)
- vue-router 4.0.6
